## 数组拆分

- 需求：按每组 10 条来拆分，最后一组不足 10 条时装剩余数据
- 使用场景：常用于对后台传来的数据进行拆分显示
- 方法：使用 slice 来拆分

```js
let data = []
for (let i = 0; i < 99; i++) {
  data[i] = i + 1
}

let arr = []
for (let i = 0; i < Math.ceil(data.length / 10); i++) {
  arr.push(data.slice(10 * i, 10 * (i + 1)))
}
console.log(arr)
```
