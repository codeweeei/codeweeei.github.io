# Java 快速入门——基础语法

## Java 程序基础

1. 第一个 Java 程序

   ```java
   public class helloword {
       /*
        * 一个程序的基本单位就是class，这里定义的class名字就是helloword（类名）
        * 类名必须以字母开头，后接字母、数字和下划线的组合，建议以大写字母开头
        * public是修饰符，表示class是公开的
        * */
       public static void main(String[] args) {
           /*
           * 这里main是方法名(建议小写字母开头)，返回值为void，表示没有任何返回值
           * public修饰符不仅可以修饰class外，还可以修饰方法，static表示静态方法
           * 在方法内部，语句才是真正的执行代码。Java的每一行语句必须以分号;结束
           * */
           System.out.println("hello,word!");
           System.out.println("北京\t天津\t上海");
       }
   }
   ```

2. 变量

   - 在 Java 中，变量分两种：基本类型的变量和引用类型的变量

   - 在 Java 中，变量必须先定义后使用，定义变量时，可以给一个初始值 `int x = 1`，不给指定初始值的话，就相当于给它指定了默认值（0）

   - 变量的一个重要特点是可以重新赋值

   - 变量不但可以重新赋值，还可以赋值给其他变量

     ```java
     int x = 1;
     //注意重复赋值时不能再使用init了，不能重复定义
     x = 2;
     ```

3. 基本数据类型

- 基本数据类型是 cpu 可以直接进行运算的类型
- 整数类型：byte、short、int、long（结尾要加 L）
- 浮点数类型：float、double
- 字符类型：char
- 布尔类型：boolean
- 计算机内存的基本结构
  - 1 字节（byte）——最小存储单元
  - 1 字节=8bit 位—>00000000~11111111（2 进制）—>0~255（10 进制）—>00~ff（16 进制）
- byte 占 1 个字节
- short、char 占 2 个字节
- int、float 占 4 个字节
- long、double 占 8 个字节

4. 数据类型的区别

- 整型

  - 对于整型类型，Java 只定义了带符号的整型，因此，最高位的 bit 表示符号位（0 表示正数，1 表示负数）

- 浮点型

  - 对于`float`类型，需要加上`f`后缀
  - 浮点数比较不能直接用`==`来比较，不精确，应该使用`Math.abs(average - 77.733333) < 0.000001`来比较

- 布尔类型

  - 布尔类型`boolean`只有`true`和`false`两个值，布尔类型总是关系运算的计算结果

- 字符类型

  - 字符类型`char`表示一个字符，Java 的`char`类型除了可表示标准的 ASCII 外，还可以表示一个 Unicode 字符

  - 注意`char`类型使用单引号`'`，且仅有一个字符，要和双引号`"`的字符串类型区分开

- 引用类型

  - 引用类型最常用的就是 String 字符串
  - `String s = "hello"`
  - 引用类型的变量类似于 C 语言的指针，它内部存储一个“地址”，指向某个对象在内存的位置

- 常量
  - 定义变量的时候，如果加上`final`修饰符，这个变量就变成了常量`final int X = 1`，常量名通常全部大写
  - 常量在定义时进行初始化后就不可再次赋值，再次赋值会导致编译错误
- 变量的作用范围
  - 变量都以`{}`作为它们自身的范围
  - 超出了作用域`{}`引用这些变量，编译器会报错

5. 运算

- &与运算、|或运算、~非运算、^异或运算
- 运算优先级->（）优先级最大

6. 常见的转义字符

- `\"` 表示字符`"`
- `\'` 表示字符`'`
- `\\` 表示字符`\`
- `\n` 表示换行符
- `\r` 表示回车符
- `\t` 表示 Tab
- `\u####` 表示一个 Unicode 编码的字符

7. 数组

- 数组变量初始化必须指明数组的大小
- 数组所有元素初始化为默认值，整型都是`0`，浮点型是`0.0`，布尔型是`false`
- 注意数组是引用类型，并且数组大小不可变

- ```java
  //数组初始化
  int[] ns = new int[5];
          ns[0] = 68;
          ns[1] = 79;
          ns[2] = 91;
          ns[3] = 85;
          ns[4] = 62;
  //数组初始化简写
  int[] ns = new int[] { 68, 79, 91, 85, 62 };
  //进一步简写
  int[] ns = { 68, 79, 91, 85, 62 };
  ```

8. IF 判断

- 判断引用类型的变量内容是否相等，不能使用`==`来进行判断，得使用`equals()`方法
- `s1.equals(s2);`，注意 s1 为 null 时会报错，可以使用`&&`与运算里的短路用法来代替`if判断`

9. 数组操作

- 遍历数组

  - 一般遍历数组使用 for 循环来遍历

  - `for each`循环，直接迭代数组的每个元素

  - ```java
    int[] ns = {1,2,3,4};
    for(int n : ns){
       //n表示ns数组元素
    }
    ```

- 数组排序

  - 冒泡排序

  - ```java
    int[] ns = {2,3,5,1,8,4};
    for(int i = 0; i<ns.length-1; i++){
          for(int j = 0; j< ns.length-i-1; j++){
                if(ns[j]>ns[j+1]){
                    //升序排列
                      int temp = 0;
                      temp = ns[j];
                      ns[j] = ns[j+1];
                      ns[j+1] = temp;
                }
          }
    }
    System.out.println(Arrays.toString(ns));//[1,2,3,4,5,8]
    ```

  - 插入排序

  - 快速排序

- 多维数组

  - 使用`Arrays.deepToString()`方法来直接取出多维数组
