Java 快速入门——基础语法

Java 程序基础

  1. 第一个 Java 程序

  2. 变量

    • 在 Java 中，变量分两种：基本类型的变量和引用类型的变量

    • 在 Java 中，变量必须先定义后使用，定义变量时，可以给一个初始值 int x = 1，不给指定初始值的话，就相当于给它指定了默认值（0）

    • 变量的一个重要特点是可以重新赋值

    • 变量不但可以重新赋值，还可以赋值给其他变量

  3. 基本数据类型

  • 基本数据类型是 cpu 可以直接进行运算的类型
  • 整数类型：byte、short、int、long（结尾要加 L）
  • 浮点数类型：float、double
  • 字符类型：char
  • 布尔类型：boolean
  • 计算机内存的基本结构
    • 1 字节（byte）——最小存储单元
    • 1 字节=8bit 位—>0000000011111111（2 进制）—>0255（10 进制）—>00~ff（16 进制）
  • byte 占 1 个字节
  • short、char 占 2 个字节
  • int、float 占 4 个字节
  • long、double 占 8 个字节
  1. 数据类型的区别

  • 整型

    • 对于整型类型，Java 只定义了带符号的整型，因此，最高位的 bit 表示符号位（0 表示正数，1 表示负数）

  • 浮点型

    • 对于float类型，需要加上f后缀
    • 浮点数比较不能直接用==来比较，不精确，应该使用Math.abs(average - 77.733333) < 0.000001来比较

  • 布尔类型

    • 布尔类型boolean只有true和false两个值，布尔类型总是关系运算的计算结果

  • 字符类型

    • 字符类型char表示一个字符，Java 的char类型除了可表示标准的 ASCII 外，还可以表示一个 Unicode 字符

    • 注意char类型使用单引号'，且仅有一个字符，要和双引号"的字符串类型区分开

  • 引用类型

    • 引用类型最常用的就是 String 字符串
    • String s = "hello"
    • 引用类型的变量类似于 C 语言的指针，它内部存储一个“地址”，指向某个对象在内存的位置

  • 常量

    • 定义变量的时候，如果加上final修饰符，这个变量就变成了常量final int X = 1，常量名通常全部大写
    • 常量在定义时进行初始化后就不可再次赋值，再次赋值会导致编译错误

  • 变量的作用范围

    • 变量都以{}作为它们自身的范围
    • 超出了作用域{}引用这些变量，编译器会报错
  1. 运算
  • &与运算、|或运算、~非运算、^异或运算
  • 运算优先级->（）优先级最大
  1. 常见的转义字符
  • \" 表示字符"
  • \' 表示字符'
  • \\ 表示字符\
  • \n 表示换行符
  • \r 表示回车符
  • \t 表示 Tab
  • \u#### 表示一个 Unicode 编码的字符
  1. 数组

  • 数组变量初始化必须指明数组的大小

  • 数组所有元素初始化为默认值，整型都是0，浮点型是0.0，布尔型是false

  • 注意数组是引用类型，并且数组大小不可变

  • //数组初始化
int[] ns = new int[5];
        ns[0] = 68;
        ns[1] = 79;
        ns[2] = 91;
        ns[3] = 85;
        ns[4] = 62;
//数组初始化简写
int[] ns = new int[] { 68, 79, 91, 85, 62 };
//进一步简写
int[] ns = { 68, 79, 91, 85, 62 };

  • 数组排序

    • 冒泡排序

    • int[] ns = {2,3,5,1,8,4};
for(int i = 0; i<ns.length-1; i++){
      for(int j = 0; j< ns.length-i-1; j++){
            if(ns[j]>ns[j+1]){
                //升序排列
                  int temp = 0;
                  temp = ns[j];
                  ns[j] = ns[j+1];
                  ns[j+1] = temp;
            }
      }
}
System.out.println(Arrays.toString(ns));//[1,2,3,4,5,8]
